# Lesson 45 - Redux Toolkit Homework

## Description

This project demonstrates global state management using Redux Toolkit and React Redux.

The application stores user data in a Redux store and allows updating the state through Redux actions.

## Technologies

- React
- TypeScript
- Vite
- Redux Toolkit
- React Redux

## Features

- Redux store configuration
- Redux Toolkit slice
- Global state management
- useSelector and useDispatch hooks
- State updates through Redux actions

## Installation

Clone the repository:

```bash
git clone <repository-link>
```

Open the project folder:

```bash
cd lesson45
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
