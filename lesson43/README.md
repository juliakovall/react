# Recipe App

## Project Description

Recipe App is a React application built with Vite and TypeScript.
The project uses React Router for page navigation and a recipe API for fetching recipe data.

The application includes:

- page routing;
- navigation with NavLink;
- recipe search;
- opening recipe details on click;
- API integration.

---

## Technologies

- React
- TypeScript
- Vite
- React Router DOM
- Fetch API
