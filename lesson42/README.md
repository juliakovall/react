# Lesson 42 - Axios Data Fetcher

## Description

This project was created with Vite, React and TypeScript.

The app includes a DataFetcher component that loads posts from the server using axios, useEffect and async/await.

## Technologies

- React
- TypeScript
- Vite
- Axios

## Features

- Fetching data from API
- Loading state
- Error state
- Displaying posts on the page

## Installation

```bash
npm install
```
