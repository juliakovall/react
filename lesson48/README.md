# Material UI React Homework

## Description

This project was created with React, TypeScript and Vite.

The main goal of the project is to integrate a UI library into a React application.
Material UI was used for creating ready-made interface components.

The project includes:

- Header component
- Form component
- Card component
- Material UI components
- Responsive and clean layout

## Technologies

- React
- TypeScript
- Vite
- Material UI
- CSS
